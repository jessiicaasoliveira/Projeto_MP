Função: Verificar as Redondezas da Tropa do PC.

Descrição: A Função recebe a struct de uma tropa e verifica as redondezas dessa tropa, e retorna um valor de acordo com o que está do lado dessa tropa.

Parâmetros:

Parameters
Troop*t - struct da tropa
intmode - flag para identificar se a função foi chamada para checar se a tropa está perto do prédio responsável por aumentar tropas (0) ou se foi chamada para checar se há tropas inimigas por perto (1)

Valor retornado:

Returns
0 - caso a tropa não esteja perto de nada
1 - caso a mode = 0 e a tropa esteja perto do prédio responsável por gerar tropas
4 - caso a tropa esteja perto do base 'B'
5 - caso a tropa esteja perto do prédio 'X'
6 - caso a tropa esteja perto do prédio 'Y'
7 - caso a tropa esteja perto do prédio 'Z'

Assertiva de entrada: Troop* t != NULL int mode == 0 || mode == 1

Assertiva de saída: A função retorna valores fixos.
