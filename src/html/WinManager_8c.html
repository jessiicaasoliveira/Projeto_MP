void startWin ( )

Função: Iniciar as funções da Ncurses usadas no jogo.

Descrição: A Função é responsável por iniciar tudo que o programa irá utilizar. Entre eles, as specials keys, as cores, o uso do mouse, captura o tamanho do terminal, desabilita o echo (a necessidade de apertar enter para capturar uma entrada) e inicia a janela da ncurses. Com tudo inicializado as outras funções podem funcionar.
